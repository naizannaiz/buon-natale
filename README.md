# Buon Natale 2025

An interactive showcase website for the Jyothi Engineering College's robotics and AI contributions to the Buon Natale 2025 festival in Thrissur.

## Features

- **Interactive 3D Santa Viewer**: Inspect the AI-Robo Santa model in 3D.
- **Innovation Showcase**: Highlights of our 4 key contributions (Santa, Drone, Tree, Float).
- **Event Countdown**: Real-time timer for the event start.
- **Mobile Responsive**: Optimized for all devices with premium glassmorphic UI.

## Tech Stack

- **Framework**: React + Vite
- **3D**: React-Three-Fiber / Three.js
- **Animations**: Framer Motion
- **Styling**: Vanilla CSS with modern design tokens

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Run development server: `npm run dev`
